La aplicacion para el reto 3 se ha realizado implementando una base de datos con la cual 
podemos ingresar tanto imagenes de los productos a ofertar en la tienda como el nombre,
una breve descripcion de los ingredientes y el precio, estos los ubicara en una lista 
donde se podran gestionar realizando acciones de actualizar, borrar o ingresar mas 
elementos.

Para el funcionamiento de la app se debe tener en cuenta que para eliminar y actualizar 
elementos de la lista se debe ingresar el numero ID.

Del mismo modo para visualizar los productos presentes en la lista se debe recargar la 
pantalla de productos.   

Se adjunta link de GitHub donde se aloja el proyecto completo junto a la APK:
https://github.com/Talamaur30/Reto3-Ciclo4.git
